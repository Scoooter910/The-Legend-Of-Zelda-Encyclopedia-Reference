Legend of Zelda Encyclopedia
Welcome to the Legend of Zelda Encyclopedia repository! This project is a personal endeavor to curate a comprehensive collection of information about the beloved Legend of Zelda series by Nintendo, integrating data from external APIs and utilizing a custom API for personal use.

Features:
External API Integration: Pulls data from external APIs to gather information about characters, items, locations, and more from the Legend of Zelda universe.

Custom API: Provides an API for personal use to access curated Zelda data programmatically, enabling seamless integration into personal projects or applications.

Interactive Interface: Explore the vast world of Hyrule through an intuitive and user-friendly interface that allows seamless navigation and discovery of information.

Continuous Updates: The repository is regularly updated with new content and features, ensuring it remains up-to-date with the latest information from the Legend of Zelda series.
